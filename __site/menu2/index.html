A bit more highlighting

Extension of highlighting for pkg an shell mode in Julia:

(v1.4) pkg> add Franklin
shell> blah
julia> 1+1
(Sandbox) pkg> resolve

you can tune the colouring in the CSS etc via the following classes:

  • .hljs-meta (for julia>)

  • .hljs-metas (for shell>)

  • .hljs-metap (for ...pkg>)
